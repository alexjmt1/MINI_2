About PyTutor

Project Information

PyTutor is an intelligent Python programming assistant developed as part of our academic project at Mangalam College of Engineering. It combines both offline natural language processing capabilities and online AI through Google's Gemini API to provide comprehensive Python programming assistance.

Key Features:

  • Interactive Python code tutoring
  • Syntax error detection and correction
  • Code debugging assistance
  • Concept explanations with examples
  • Both online and offline capabilities
